Шифр Цезаря v1.0 (beta) 12/12/218

Данное классическое консольное приложение позволяет работать с кириллическим текстом
для шифрования и дешифрования его методом простой перестановки с постоянным шагом сдвига (метод Цезаря).

В программе реализованы следущие функции, которые вы можете выбрать из меню:

- Шифрование введенного в консоль текста методом Цезаря;
- Дешифрование текста из txt-файла;
- Вывод результатов шифрования/дешифрования на экран или запись их в файл.

Для шифрования или дешифрования текста вым требуется указать ключ. Ключом является число в пределах от -33 до 33, на которое
смещается исходный текст при шифровании или дешифровке относительно алфавита. Отрицательный ключ - сдвиг влево, положительный
ключ - сдвиг вправо. 

Для записи результата в файл требуется указать директорию и имя файла в формате "[диск]:\[путь]\[имя_файла.txt]".

Основной функционал приложения покрыт автоматическими Unit тестами, с использованием стандартных возможностей Unit Testing Framework. 
Для динамического определения кодировки текстового файла подключена сторонняя библиотека UDE.CSharp 1.1.0.
